# Changelog

All notable changes to **626 Labs Themes** are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project uses Conventional Commits.

## [0.1.0] — 2026-04-29

### Added

- feat: initial release with two variants — **626 Labs Night** (deeper) and **626 Labs Storm** (lighter)
- feat: full editor chrome — editor, sidebar, tabs, status bar, panel, terminal, peek view, diff editor, notebook, chat, command center
- feat: full TextMate syntax coverage modeled on Tokyo Night's scope tree, with the 626 Labs cyan-magenta duo carrying logic and control flow
- feat: semantic token coloring for TypeScript / JavaScript types, classes, parameters, decorators
- feat: terminal ANSI palette mapped to the brand semantic colors (cyan, magenta, green, amber, red)
- feat: bracket pair highlighting cycles through the full brand palette
- feat: git decorations and gitlens gutter colors tuned for the dark navy field
